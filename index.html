DIG 245 View Source


Viewing the source code of a web page the short way in Microsoft Edge :

  1. Ctrl + Shift + I
  2. Click on the elements tab and view the source code

Viewing the source code of a web page the long way in Microsoft Edge :

  1. Right click on the page
  2. Click Inspect
  3. Go to the elements tab to view the source code

Some other things you can do with dev tools besides see HTML and CSS code

  • You can see the file structure and the different files that the web page uses. This allows you to see what materials were needed to create the webpage.
  • You can also see the memory usage of the current site, including the total JS heap size.
  • Finally, you can see a breakdown of the page's performance. This includes metrics on how much time the site has spent on scripting, rendering, painting, system, or idle.
